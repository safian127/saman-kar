# سامان کار

یک شروع‌کننده‌ی اندرویدی فارسی برای مدیریت کارها با حال‌وهوای GTD، الهام‌گرفته از تجربه‌ی Everdo اما با نام، ساختار و پیاده‌سازی مستقل.

## قابلیت‌های نسخه‌ی اولیه

- رابط فارسی و راست‌به‌چپ
- لیست‌های صندوق، اقدام بعدی، زمان‌بندی و روزی شاید
- افزودن، ویرایش، حذف و انجام‌شده کردن کارها
- پروژه/حوزه و یادداشت برای هر کار
- تاریخ هجری شمسی بدون وابستگی خارجی
- ذخیره‌سازی آفلاین با `SharedPreferences`

## اجرا

1. پوشه را در Android Studio باز کنید.
2. صبر کنید Gradle Sync کامل شود.
3. روی Run بزنید و یک Emulator یا گوشی اندرویدی انتخاب کنید.

## ساخت APK بدون Android Studio با GitHub

1. در GitHub یک repository جدید بسازید.
2. فایل‌های این پروژه را داخل repository آپلود کنید.
3. به تب `Actions` بروید.
4. workflow با نام `Build Android APK` را باز کنید.
5. اگر خودکار اجرا نشد، دکمه‌ی `Run workflow` را بزنید.
6. بعد از سبز شدن build، پایین صفحه بخش `Artifacts` را باز کنید.
7. فایل `saman-kar-debug-apk` را دانلود کنید. داخل آن `app-debug.apk` قرار دارد.

برای نصب روی گوشی، ممکن است لازم باشد گزینه‌ی نصب از منابع ناشناس را برای مرورگر یا فایل‌منیجر فعال کنید. این فایل debug است و برای تست شخصی مناسب است؛ برای انتشار رسمی در کافه‌بازار یا Google Play باید نسخه‌ی release امضا شده بسازیم.

اگر خواستید پروژه جدی‌تر شود، قدم‌های بعدی پیشنهادی این‌ها هستند: پایگاه‌داده Room، تقویم شمسی تصویری، فیلترهای کانتکست/برچسب، جست‌وجو، بکاپ، و همگام‌سازی.
